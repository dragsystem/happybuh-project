Descripción

ChangeGame consiste en encontrar la imagen que se muestra al principio entre otras muchas mostradas.

Cada vez que empieza la partida las imágenes cambian de posición

Instrucciones

El usuario deberá hacer click con el dedo en la imagen correcta. Éste tendrá tres oportunidades para acertar.

Por cada imagen encontrada el usuario gana una moneda y 0.1 de experiencia.

En el borde izquierdo superior se encuentra el contador de monedas obtenidas durante la partida y el número de vidas restantes.

Cuando el contador de vidas llegue a 0 la partida se acabará y el usuario podrá decidir si repetir experiencia o cambiar de juego. En este caso sólo hay una vida, así que mejor tener cuidado de no caerse.

Por otro lado, en el borde derecho superior se encuentra el botón para pausar la partida en un momento dado que el usuario lo requiera. Keep calm! aunque el usuario decida salir de la partida las monedas y experiencia obtenidas se sumarán igualmente.
